# RAG SaaS Backend

A FastAPI backend for a Retrieval-Augmented Generation (RAG) platform.

## Features

- User Authentication (Supabase)
- Workspace Management
- Document Upload & Processing
- Vector Embeddings
- API Key Management
- AI Chat Integration
- Usage Tracking

## Tech Stack

- FastAPI
- Supabase
- Pinecone
- OpenAI
- Python
